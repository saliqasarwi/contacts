# Contacts (React Router 6 Tutorial)

This is a simple **Contacts** application built while following the official **[React Router v6.28 Tutorial](https://reactrouter.com/6.28.0/start/tutorial)**.
## Features

- Add new contacts
- Edit existing contacts
- Delete contacts
- View contacts details
- Search contacts
- Error handling with custom error pages
- Clean nested UI with shared layout

## Tech Stack

- React 19.2
- React Router v6.28.0 
- Vite 
- JavaScript 

## How to Run Locally

```bash
# Clone the repository
git clone https://github.com/saliqasarwi/contacts.git
cd contacts

# Install dependencies
npm install

# Start the development server
npm run dev
Open http://localhost:5173 to view it in the browser.
